# GitHub Pro
## When to Activate
- GitHub repos, PRs, issues, Actions, releases
- Git operations (branch, merge, rebase, tag)
## Quick Reference
- gh auth login/status
- gh repo create/clone/view
- gh pr create/list/checks/merge
- gh issue create/list/close
- gh run list/view/watch/rerun
- gh release create/list
- gh api for REST/GraphQL queries
## Safety
Confirm before force-push. Never commit secrets. Always branch.
