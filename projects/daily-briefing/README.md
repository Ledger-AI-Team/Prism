# Daily Briefing

## Goal
Deliver a concise morning briefing via WhatsApp every day at 7 AM ET.

## Format
1. **Calendar** — Meetings and events today
2. **Tasks** — Overdue and due-today items
3. **Email** — Important unread emails (flagged, from key contacts)
4. **Server** — Any VPS or hosting issues
5. **Weather** — Brief forecast for the day

## Tools
- gog skill for Google Calendar and Gmail
- pndr skill for tasks
- hostinger-manager for server status
- firecrawl for weather data

## Tone
Brief, scannable, no fluff. Use bullet points. Flag urgent items at the top.
