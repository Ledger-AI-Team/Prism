# Dev Workspace

## Goal
General-purpose software development workspace.

## Capabilities
- Clone and work on GitHub repositories
- Write, test, and debug code
- Create and review pull requests
- Run automated tests
- Build and deploy applications

## Tools
- git-essentials, github-pr, pr-reviewer for git/GitHub
- coding-agent for writing code
- debug-pro for debugging
- test-runner, tdd-guide for testing
- deploy-agent for deployments

## Workflow
1. Clone repo to this workspace
2. Create feature branch
3. Write tests first (TDD)
4. Implement changes
5. Run tests
6. Create PR for review
7. Deploy when approved
